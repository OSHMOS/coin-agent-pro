# CoinAgent Pro

코인원(Coinone) 자동매매 봇 (Electron 기반 데스크탑 앱)

## 주요 기능
- 코인원 API를 연동한 실시간 자산 조회 및 자동 매수/매도 시스템
- 데스크탑 앱(Mac/Windows) 환경에서 직관적인 UI(GUI) 제공
- 목표 수익률 도달 시 자동 익절 (Take Profit)
- RSI, 볼린저 밴드 등 시장 보조 지표 등에 기반한 로직 커스터마이징 가능

---

## 🚀 시작하기 전에: 코인원 API 발급 방법

본 자동매매 봇을 실행하려면 회원님의 코인원 계정에서 발급한 **Access Token**과 **Secret Key**가 필요합니다. 보안상 자산 출금 등의 권한은 제외하고 최소한의 거래 권한만 주어야 합니다.

### 1단계: API KEY 생성
1. [코인원(Coinone)](https://coinone.co.kr/) 접속 및 로그인 진행
2. 홈페이지 우측 상단 프로필 이미지 클릭 ➞ **[마이페이지]** ➞ **[API 관리]**로 이동
3. "API 등록(새로운 API 생성)" 버튼을 클릭
4. 가장 중요한 **권한 설정** 단계입니다. **반드시 다음 두 개만 활성화**하세요:
   - ✅ **잔고조회** (자산 확인용)
   - ✅ **주문하기** (매수/매도용)
   - 🚨 **출금하기** (절대 활성화하지 마세요! 보안상 매우 위험합니다)
5. 인증 절차(OTP 등)를 거쳐 생성을 완료합니다.
6. 완료 화면에 **Access Token**과 **Secret Key**가 나타납니다.
   - **(주의) Secret Key는 이때 단 한 번만 화면에 표시됩니다!** 창을 닫기 전에 메모장 등에 안전하게 복사해 두시기 바랍니다.

---

### 2단계: 환경변수(.env) 파일 설정

1. 프로젝트 폴더 최상단(루트)에 `.env` 라는 이름의 새 파일을 만듭니다. (※ 절대로 이 파일을 깃허브 등에 업로드하지 마세요. `.gitignore`에 자동 포함되어 있습니다.)
2. 아래 내용을 복사하여 `.env` 파일에 붙여넣고, 발급받은 키 값과 봇 구동 옵션을 작성합니다.

```env
# ====== [필수] API 설정 ======
COINONE_ACCESS_TOKEN=방금_발급받은_액세스_토큰을_여기에_붙여넣으세요
COINONE_SECRET_KEY=방금_발급받은_시크릿_키를_여기에_붙여넣으세요

# ====== [선택] 봇 세부 동작 설정 ======
BOT_TARGET_COINS=BTC,ETH,SOL    # 거래 대상 암호화폐 (쉼표로 구분)
BOT_BUY_AMOUNT_KRW=10000        # 1회 매수 금액 (원)
BOT_CHECK_INTERVAL_MIN=15       # 시장 상황 체크 주기 (분 단위)
BOT_TAKE_PROFIT=2.0             # 목표 익절 수익률 (%)
BOT_STOP_LOSS=0                 # 손절 라인 (%) - 0입력 시 사용 안 함
BOT_RSI_BUY=30                  # 매수에 사용할 RSI 기준선
BOT_RSI_SELL=70                 # 매도에 사용할 RSI 기준선
BOT_MAX_DAILY_TRADES=10         # 일일 최대 거래 가능 횟수
BOT_COOLDOWN_HOURS=1            # 거래 후 휴식 시간 (시간 단위)
```
*(기본 설정값을 그대로 유지해도 작동되지만, 본인 투자 성향에 맞게 자유롭게 수정하세요.)*

## 💻 로컬에서 설치 및 실행 방법

1. 소스 코드를 다운로드하고 해당 폴더(디렉토리)로 진입합니다.
2. 종속성 모듈(Node.js 패키지)을 설치합니다.
   ```bash
   npm install
   ```
3. 편리한 UI와 봇 구동이 같이 되는 **데스크탑 앱(GUI)** 버전을 띄우려면:
   ```bash
   npm run gui
   ```
4. 터미널 백그라운드에서 **봇 프로세스만** 돌리고 싶다면:
   ```bash
   npm run bot
   ```

## 🛠 실행 파일(설치/포터블) 빌드 방법
본인의 데스크탑 환경에 맞게 직접 실행파일(.exe, .dmg 등)로 빌드할 수 있습니다.

- **Mac용 빌드 (.dmg)**
  ```bash
  npm run build:mac
  ```
- **Windows용 빌드 (.exe)** (※ Mac에서 윈도우용으로 빌드 시 호환을 위해 `--x64` 옵션을 붙여야 할 수 있습니다)
  ```bash
  npx electron-builder --win --x64
  ```
빌드가 완료되면 `/dist` 폴더 내에 설치 파일이 생성됩니다.

## ⚠️ 주의사항 및 면책 조항 (필독)
1. **API 키 관리**: 시크릿 키는 절대 타인에게 공유하거나 공개 공간(GitHub, 블로그 등)에 유출하지 마세요. 실수로라도 `.env` 파일이 업로드되지 않도록 주의해야 합니다.
2. **원금 손실 위험**: 암호화폐 시장은 24시간 변동성이 매우 큽니다. 봇이 항상 수익을 보장하지 않으므로, 사용으로 인해 발생하는 어떠한 금전적 손실에 대해서도 개발자는 책임지지 않습니다. 충분한 소액 테스트를 거친 뒤 사용하시기 바랍니다.
